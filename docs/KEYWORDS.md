# Keywords for PLOS ONE Submission

Last updated: 2026-05-29

## Proposed Keywords (8)

1. **Swedish Riksdag** — geographic/institutional scope
2. **parliamentary text analysis** — domain + method
3. **multimodal political measurement** — core methodological contribution
4. **party ideology estimation** — substantive topic
5. **speech-action consistency** — second substantive topic (say-do alignment)
6. **deterministic classifier** — methods characterization (rule-first, reproducible)
7. **natural language processing** — computational method class
8. **political text data** — data type

## Rationale

- Covers the methodological niche (NLP + deterministic rules applied to parliamentary data).
- Covers the empirical domain (Swedish parliament, party ideology, say-do consistency).
- Four keywords are method-focused, four are domain-focused — balanced for interdisciplinary discovery.
- All terms are commonly used in political science and computational social science literature, improving discoverability.

## PLOS ONE Submission Note

Enter these as a comma-separated list in the PLOS ONE submission system's "Keywords" field. PLOS ONE accepts up to 10 keywords; 8 is appropriate for this manuscript's scope.
